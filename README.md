# Client

Sử dụng Angular phiên bản 13.3.6.

## Install package

`npm install`

## Development server

`ng serve` hoặc `npm start`

## Build

Chạy `ng build` để build project. Dữ liệu sau khi build được lưu tại `dist/`.

## Running unit tests

Chạy `ng test` để chạy unit test bằng [Karma](https://karma-runner.github.io).
